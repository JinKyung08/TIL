네이버 챗봇 정규식   

- 필수 발화 표현식 [ ]

- 선택 발화 표현식 ( )

- 임의의 단어를 매칭하는 표현식 <?>
	<?> 길동
	-> 홍길동 (O) / 홍홍길동 (X) 
	[안녕] <?>*?
	-> 안녕 철수야 (O) / 안녕 (O)
	[안녕] <?>+?
	-> 안녕 철수야 (O) / 안녕 (X)
